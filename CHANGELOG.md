# ABDL Space 更新日志

## v5.2.0 — 2026-05-09

### 🐛 Bug 修复
- **纸尿裤列表为空修复**：简化首页加载逻辑，移除有问题的 debounce 复杂度，确保首次加载正常显示
- **readAllNotifications 错误修复**：`Promise.resolve()` 替代空函数，不再导致 `.catch()` 崩溃
- **侧边栏文字高亮修复**：添加 `user-select: none` 和 `-webkit-tap-highlight-color: transparent`，点击不再出现蓝色背景
- **按钮颜色增强**：禁用按钮现在有更明显的灰色滤镜 + 降低透明度，避免被误认为不可交互

### ✨ 新功能

#### 对比工具整合进纸尿裤页面 (#5)
- 首页新增「对比模式」切换按钮
- 在纸尿裤卡片上直接勾选 2-4 款进行对比
- 对比结果以内联表格 + 雷达图展示，无需跳转独立页面
- 独立 `/compare` 路由保留

#### 通知整合进私信 (#3)
- 私信页面新增「系统通知」会话入口
- 通知以只读消息形式展示（无输入框）
- 未读通知红点计数
- 点击通知可跳转到对应帖子

#### 键盘快捷键 (#8)
- `Ctrl+Shift+T` 切换深色/浅色模式
- `Alt+1` ~ `Alt+9` 快速导航各页面
- `Ctrl+D` 跳转纸尿裤列表
- 输入框内自动禁用快捷键

### 📱 移动端优化
- 搜索栏竖向排列、select 全宽
- 私信布局竖向堆叠
- 按钮最小触控尺寸 44px/36px
- 主题切换按钮缩小避免遮挡
- 表格横向滚动
- `.hide-mobile` 辅助类隐藏移动端次要文字

### 🔐 安全
- API key 改用 `VITE_DEEPSEEK_KEY` 环境变量
- 已从全部 git 历史中清除明文 key
